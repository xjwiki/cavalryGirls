战略支援(空中打击)

01. 基本信息

01. 目前的战略武器共有 20 种, 它们的功能多种多样, 有的可杀伤敌人, 有的则侧重于削弱, 甚至纳米修复机器人还能恢复机甲 HP;
02. 整备城防界面默认有 2 个战略武器的安装槽位, 可用资金解锁更(最多 7 个), 费用为 15+N, N 是当前已经解锁的槽位数;
03. 有些战略武器是无法被合成的, 它们是驾驶员忠诚度到达某个级别的奖励, 只有先获得过 1 件以解锁后, 采购部里才可能会刷出;
04. 战略武器不像机甲配件, 它们既可以被重复安装, 在战斗中也可以被重复使用, 但使用会扣除战后的收益, 细节可查阅 # 额外拓展;

02. 额外拓展

调用方法: 于整备城防的界面, 将战略武器安装进槽位之后, 在战场即可通过按 Tab 调出战略界面使用, 或是通过快捷键 1~7 调用, 如果安装时跳过了某个槽位, 则布局会忽视被跳过的槽位, 例如第四个槽位放空, 则第五个槽位的战略武器会填补这个空缺, 既五变成四;

费用计算: 战略武器的费用由 BaseSalary.csv 表中的 UseStrategyCostRate 属性决定, 该属性的五个值为 -2, -1, -2, -2, 0, 分别代表了, 故事/正常, 硬核/残酷, 挑战模式, 无尽模式, 潜入模式, 具体的算法为 CD × 0.01 × UseStrategyCostRate, 例如在故事/正常模式下, 饱和轰炸的 CD 为 50(秒), 那使用一次饱和轰炸的价格就是 50 × 0.01 × -2 = -1, 战后于战略武器费用中扣除;

天气影响: 天气会影响战略武器的 CD, 算法为 RainRate × WeatherAffectCDRate ÷ 100 + 1, RainRate 是场景阴晴状态的随机数, 细节可查阅: 剧情模式.html / 特殊遭遇.html, WeatherAffectCDRate 是 Base.csv 表中的参数, 默认预设为 2, 例如当前场景的 RainRate 为 60, 则 CD 为 60×2÷100+1=2.2, 假设调用饱和轰炸, CD 就是 50×2.2=110 秒, 但只有阴天既 RainRate > 0 才有影响, 如果是晴天则对 CD 没影响, 而且天气也不影响费用的计算(玛丽的 LV4 忠诚度奖励可 -20% CD, 但也不影响费用的计算);

————

解锁时间: 在剧情模式下, 战略武器的开放时间由变量 $StrategyOpenDay 控制, 该变量代表着战略武器的开放事件, 而这个事件被放在了第 22 天, 而部分战略武器的解锁天数也是由这个变量控制, 因此它们的解锁天数也是 22, 此外, 黑夜战场的开放时间由变量 $NightOpenDay 控制, 而这个事件被放在了第 65 天, 空投照明弹的解锁天数也是由这个变量控制, 因此它的解锁天数也就是 65;
