编队建制(搭配成群)

01. 基本信息

01. 敌人基本上都会以某种特定的混编形式出现在战场上, 而下表就是所有可能出现的组合, 适配于不同的场景模式, 目前共计有 500
02. 实际上大多数的场景地图, 都拥有复数的组合选项, 遭遇后会从选项中随机选出某个来用, 所以即便是同个场景, 敌人也未必会相同
03. 某个场景具体会出什么敌人组合以及会出的数量, 均由场景数据控制, 例如想了解剧情模式下的敌人构成, 可以查阅 剧情模式.html

项目序号 项目代码 敌人注解 敌人组合

02. 额外拓展

特殊组合: 代码 1000 ~ 1016 和 B4 这 18 个项, 会根据场景数据的 MapType 和 EnemyType 属性变化, 当 MapType 属性不为空且 EnemyType 为 [A|B|C|D] 任意值时, 具体敌人的代码就会在尾部加上 A 或 B 或 C 或 D, 直到 MapType 属性为 0 或 EnemyType 属性为 none 或 none|none, 总而言之, 就是这些项的敌人, 会根据场景的数据变化(所以图片也是动态的)
