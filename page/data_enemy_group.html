01. 基本信息

01. 敌人基本上都会以某种特定的混编形式出现在战场上, 而下表就是所有可能出现的组合, 适配于不同的场景模式, 目前共 438 种;
02. 可将这些敌人组合分为"剧情/无尽/挑战/潜入"四大类, 但有些组合在别的模式下也可能会出现, 所以就只是种粗略的场景划分而已;
03. 实际上大多数的场景地图, 都拥有复数的组合选项, 遭遇后会从选项中随机选出某个来用, 所以即便是同个场景, 敌人也未必相同!;
04. 某个场景具体会出什么敌人组合以及会出的数量, 均由场景数据控制, 例如想了解剧情模式下的敌人构成, 可查阅: 剧情遭遇.html;

02. 额外拓展

装饰图片: 你可以已经发现了, 在敌人组合中有张奇怪的 None 图片, 就是没有敌人缩略图, 仅有深棕色的地面背景且带个白色的 X, 这个是 XJ 为了排版匀称美观而添加的装饰性图片, 实际上就只是个占位符而已, 并没有任何实质上的意义, 强迫症发作的产物罢了(T_T);

特殊组合: 实际上组合代码为 1000 ~ 1016 和 B4 这共计有 18 个的选项, 它们都会根据场景数据的 MapType 和 EnemyType 属性进行变化, 当 MapType 属性不为空且 EnemyType 为 A|B|C 或 A|B 或 A|C 或 B|C 时, 具体敌人的代码就会在尾部加上 A 或 B 或 C, 直到遇见 MapType 属性为 0 或 EnemyType 属性为 none 或 none|none 为止, 总而言之就是这里的敌人存在三种可能性, 会根据场景数据而变化, 但这里放动态图会显得很不和谐, 所以最后表格只能暂时以 A 尾缀的敌人缩略图代替;
